Напишите декоратор limit_calls, который:

Ограничивает количество вызовов функции за день

Сохраняет счётчик вызовов в файл call_counter.txt

Формат файла: дата:количество_вызовов

Если лимит превышен, выводит сообщение и не выполняет функцию

Параметры декоратора:

@limit_calls(max_calls=3)
def some_function():
    print("Function executed")

Требования:

Используйте модуль datetime для получения текущей даты

Читайте и обновляйте файл call_counter.txt

Если дата в файле не совпадает с текущей, сбрасывайте счётчик

Декоратор должен принимать параметр max_calls
