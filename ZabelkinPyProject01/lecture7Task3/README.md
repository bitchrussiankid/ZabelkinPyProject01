Создайте файл poem.txt со следующим содержимым:

Roses are red,
Violets are blue,
Python is fun,
And so are you!

Напишите программу, которая:

Открывает файл poem.txt для чтения

Подсчитывает количество строк в файле

Выводит результат в формате: "The file contains X lines"

Также выводит каждую строку с ее номером в формате: "Line 1: Roses are red,"

Требования:

Используйте контекстный менеджер with

Примените метод readlines() или итерацию по файлу

Учтите, что последняя строка может не иметь символа переноса
