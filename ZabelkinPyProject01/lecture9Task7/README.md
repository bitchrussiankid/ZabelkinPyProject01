Создайте класс Library (Библиотека), который будет использовать класс Book из предыдущих задач.

Требования:

Класс Library должен иметь:

Конструктор, инициализирующий пустой список books

Метод add_book(book) для добавления книги в библиотеку

Метод display_all_books() для вывода информации о всех книгах

Метод count_books() для возврата количества книг в библиотеке

Используйте класс Book с конструктором и методом display_info() из задачи 3

В основной программе:

Создайте несколько объектов Book

Создайте объект Library

Добавьте книги в библиотеку

Выведите информацию о всех книгах и их количество

Пример вывода:

Library contents:
1. 1984 by George Orwell
2. To Kill a Mockingbird by Harper Lee
3. The Great Gatsby by F. Scott Fitzgerald
Total books: 3
