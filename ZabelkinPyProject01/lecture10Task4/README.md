**Задача 4. Иерархия наследования (три уровня)**

Создай **иерархию классов** для представления транспортных средств:

1. **Базовый класс** `Vehicle` (Транспортное средство):
   - Атрибуты: `brand` (марка), `year` (год)
   - Метод: `start_engine()` — возвращает `"Двигатель запущен"`

2. **Дочерний класс** `Car` (Автомобиль), наследует от `Vehicle`:
   - Добавляет атрибут: `doors` (количество дверей)
   - **Переопределяет** `start_engine()` — возвращает `"Автомобильный двигатель запущен"`

3. **Внучатый класс** `ElectricCar` (Электромобиль), наследует от `Car`:
   - Добавляет атрибут: `battery_capacity` (ёмкость батареи в кВт·ч)
   - **Переопределяет** `start_engine()` — возвращает `"Электродвигатель запущен бесшумно"`

**Требования:**
- Каждый класс вызывает конструктор родителя через `super().__init__()`
- Продемонстрируй все три уровня наследования

**Пример:**
```python
vehicle = Vehicle("Generic", 2020)
car = Car("Toyota", 2022, 4)
electric = ElectricCar("Tesla", 2023, 4, 100)

print(vehicle.start_engine())  # Двигатель запущен
print(car.start_engine())      # Автомобильный двигатель запущен  
print(electric.start_engine()) # Электродвигатель запущен бесшумно
```

**Подсказка:** Конструктор `ElectricCar` должен принимать все параметры и передавать нужные в родительский конструктор.
