**Задача 1. Базовое наследование**

Создай **родительский класс** `Vehicle` (Транспортное средство):
- Атрибуты: `brand` (марка), `year` (год выпуска)
- Метод: `display_info()` — возвращает строку `"Марка, Год"`

Создай **дочерний класс** `Car` (Автомобиль), который наследует от `Vehicle`:
- Добавляет атрибут: `doors` (количество дверей)
- **Переопределяет** метод `display_info()` — должен возвращать `"Марка, Год, X дверей"`

**Требования:**
1. Используй синтаксис наследования: `class Car(Vehicle):`
2. В конструкторе `Car` используй `super().__init__()` для вызова конструктора родителя
3. Создай объекты обоих классов и продемонстрируй разницу в работе методов

**Пример:**
```python
vehicle = Vehicle("Generic", 2020)
car = Car("Toyota", 2022, 4)

print(vehicle.display_info())  # Generic, 2020
print(car.display_info())      # Toyota, 2022, 4 дверей
```

**Подсказка:** Конструктор `Car` должен принимать `brand, year, doors` и передавать первые два в родительский конструктор.
