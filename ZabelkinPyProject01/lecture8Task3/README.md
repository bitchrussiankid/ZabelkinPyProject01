Напишите функцию copy_file(source, destination), которая:

Принимает два аргумента: имена исходного и целевого файлов

Копирует содержимое из source в destination

Использует контекстный менеджер with для обоих файлов

Возвращает True, если копирование успешно, и False, если возникла ошибка

Обрабатывает основные ошибки: файл не найден, проблемы с доступом

Дополнительно:

Если целевой файл уже существует, он должен быть перезаписан

Выводите информационные сообщения о процессе

Требования:

Используйте try-except для обработки FileNotFoundError, PermissionError

Читайте файл построчно для эффективности

Гарантируйте закрытие файлов даже при ошибках
