Напишите программу, которая:

Создает новый файл с именем shopping_list.txt

Записывает в него 5 разных продуктов для покупки (каждый продукт с новой строки)

Гарантирует, что файл создается заново каждый раз при запуске программы (перезаписывается, если уже существует)

После записи выводит сообщение: "Shopping list created successfully!"

Дополнительное условие:

Продукты должны быть разными (не повторяться)

Каждый продукт должен быть записан с новой строки

Используйте список продуктов в коде программы

Требования:

Используйте режим 'w' для записи (создания/перезаписи файла)

Примените контекстный менеджер with

Убедитесь, что каждый продукт записан с новой строки
