Создайте файл config.txt со следующим содержимым:

host=localhost
port=8080
debug=true
username=admin
timeout=30

Напишите программу, которая:

Читает файл config.txt построчно

Парсит каждую строку, разделяя ее по знаку = на ключ и значение

Сохраняет результат в словарь config_dict

Преобразует значения к правильным типам:

Числа → int

true/false → bool

Остальное → str

Выводит на экран:

Полученный словарь с типами значений

Значение конкретного параметра (например, port)

Пример ожидаемого вывода:

{'host': 'localhost', 'port': 8080, 'debug': True, 'username': 'admin', 'timeout': 30}
Port: 8080

Требования:

Используйте контекстный менеджер with

Обрабатывайте строки, пропуская пустые строки

Учитывайте, что в файле могут быть пробелы вокруг =
