Напишите декоратор backup_result, который:

При каждом успешном выполнении функции сохраняет её результат в файл backup_results.txt

Формат записи: дата | имя_функции | аргументы | результат

Декоратор должен добавлять записи, а не перезаписывать файл

Если при выполнении функции возникает исключение — записывает в файл ошибку вместо результата

Основная функция:

@backup_result
def calculate_expression(expression):
    """Вычисляет математическое выражение (строку)"""
    return eval(expression)  # осторожно с eval в реальных проектах!

Требования:

Используйте модули datetime и traceback для деталей ошибок

Записывайте и позиционные, и именованные аргументы

Обрабатывайте любые исключения в функции
