**Задача 1. Базовое наследование (Person → Student)**

Создай **родительский класс** `Person` (Человек):
- Атрибуты: `name` (имя), `age` (возраст)
- Метод: `introduce()` — возвращает `"Меня зовут [имя], мне [возраст] лет"`

Создай **дочерний класс** `Student` (Студент), который наследует от `Person`:
- Добавляет атрибут: `student_id` (номер студенческого билета)
- **Переопределяет** метод `introduce()` — должен возвращать `"Я студент [имя], мой ID: [student_id]"`

**Требования:**
1. Используй `super().__init__()` в конструкторе `Student`
2. Создай по одному объекту каждого класса

**Пример:**
```python
person = Person("Анна", 25)
student = Student("Иван", 20, "S12345")

print(person.introduce())   # Меня зовут Анна, мне 25 лет
print(student.introduce())  # Я студент Иван, мой ID: S12345
```
