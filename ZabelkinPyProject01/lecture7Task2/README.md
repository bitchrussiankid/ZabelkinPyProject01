У вас есть файл notes.txt с начальным содержимым:

My notes:
1. Buy milk
2. Call mom

Напишите программу, которая:

Проверяет существование файла notes.txt

Если файл не существует, создает его с начальным содержимым (как выше)

Добавляет в конец файла новую строку: 3. Learn Python

Выводит обновленное содержимое файла на экран

Требования:

Используйте режим добавления 'a' для дописывания в файл

Используйте режим чтения 'r' для вывода содержимого

Применяйте контекстный менеджер with для обоих операций

Проверяйте существование файла перед работой с ним
