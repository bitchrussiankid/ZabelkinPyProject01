У вас есть два файла:

file1.txt с содержимым:

Hello from file 1
This is line 2
End of file 1

file2.txt с содержимым:

Hello from file 2
This is line B
End of file 2

Напишите программу, которая:

Создает оба файла (если они не существуют) с указанным содержимым

Объединяет содержимое обоих файлов в новый файл merged.txt

В merged.txt сначала должно быть содержимое file1.txt, затем file2.txt

Между содержимым файлов добавьте разделительную строку: "--- Merged ---"

Выводит на экран:

Сообщение "Files merged successfully!"

Содержимое результирующего файла merged.txt

Требования:

Используйте контекстный менеджер with для всех файлов

Читайте файлы построчно или целиком (на ваш выбор)

Убедитесь, что файл merged.txt создается/перезаписывается при каждом запуске
