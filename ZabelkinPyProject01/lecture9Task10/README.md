Цель: Научиться создавать взаимодействие между объектами разных классов.

Класс Student:

__init__(self, name, age) — имя, возраст

grades (список оценок, инициализируется пустым)

add_grade(self, grade) — добавить оценку в список

get_average(self) — вернуть средний балл

Класс Teacher:

__init__(self, name, subject) — имя, предмет

add_grade_to_student(self, student, grade) — добавить оценку ученику

Требования:

Создай:

Учеников: student1 ("Алиса", 15), student2 ("Боб", 16)

Учителя: teacher ("Мистер Смит", "Математика")

Учитель ставит:

Алисе: 5, 4, 5

Бобу: 3, 4

Выведи средний балл каждого ученика
