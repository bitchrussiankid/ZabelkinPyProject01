Написать функцию, которая будет работать со списком в формате dictionary (по аналогии с JSON, переданным по API) по следующим условиям:

1. JSON передает список товаров и их количество в формате {1: "10 apples"} и так далее. Таким образом, тело запроса может содержать любое количество позиций из любых наименований, при этом каждая позиция пронумерована в порядке от 1 до N.
2. Наименования и количество товаров передаются в формате string в виде единой неразделенной строки, где первая часть - количество, вторая - наименование.
3. Количество может быть не указано.

Требования:

1. Функция должна уметь отличать параметр "количество" от параметра "наименование" и делить в соответствии с этим различием единую строку на две части.
2. Функция должна работать только с разрешенным списков товаров. Список товаров должен быть легко дополняем.
3. Функция должна уметь помещать пришедшие в JSON разрешенные товары в список "inventory" в формате dictionary в "ключ", а количество товаров - в "значение" в формате {"apples": 10}.
4. Функция должна уметь складывать одинаковые товары в инвентаре в случае, если один и тот же товар пришел в JSON 2 и более раз.
5. Функция должна быть гибкой и уметь работать с любым количеством товаров, а также корректно обрабатывать товары, не входящие в разрешенный список товаров
6. Функция должна уметь работать с типичными ошибками
