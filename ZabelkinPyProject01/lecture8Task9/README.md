Напишите функцию create_data_file(filename, data_list), которая:

Принимает имя файла и список данных (строки или числа)

Создаёт новый файл с указанным именем (перезаписывает, если существует)

Записывает каждый элемент списка в файл:

Каждый элемент с новой строки

Если элемент — число, преобразует его в строку

После записи выводит статистику:

Количество записанных элементов

Типы записанных данных (сколько строк, сколько чисел)

Требования:

Используйте контекстный менеджер with

Обрабатывайте разные типы данных в списке

Возвращайте кортеж: (total_count, string_count, number_count)

Пример:

data = ["apple", "banana", 42, "orange", 3.14, 100]
result = create_data_file("output.txt", data)
# Output:
# File 'output.txt' created successfully!
# Total items: 6
# Strings: 3, Numbers: 3
# Return: (6, 3, 3)
