Создайте словарь student, содержащий информацию о студенте со следующими полями:

name (строка)

age (целое число)

course (строка)

gpa (дробное число)

is_graduated (логическое значение)

subjects (список из 3 строк - названий предметов)

Требования:

Все значения должны быть разных типов данных как указано выше

Используйте осмысленные примеры данных

Выведите созданный словарь на экран

Выведите тип данных для каждого значения в словаре
