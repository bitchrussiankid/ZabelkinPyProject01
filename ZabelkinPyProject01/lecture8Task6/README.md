Напишите декоратор cache_to_file, который:

Кеширует результат функции в файл

При первом вызове функции с определёнными аргументами:

Выполняет функцию

Сохраняет результат в файл cache.txt в формате: аргументы:результат

При повторном вызове с теми же аргументами:

Читает результат из файла (не вычисляя заново)

Возвращает сохранённый результат

Файл cache.txt должен создаваться автоматически

Тестовая функция:

@cache_to_file
def expensive_operation(x, y):
    print("Выполняю сложные вычисления...")
    time.sleep(2)  # Имитация долгой операции
    return x * y + 100

Требования:

Используйте контекстный менеджер with для работы с файлом

Ключ кеша: строка вида f"{args}_{kwargs}"

Результаты хранить в формате: ключ:значение\n
