Дан словарь:

car = {
    'brand': 'Toyota',
    'model': 'Camry', 
    'year': 2020,
    'color': 'blue'
}

Напишите программу, которая:

Запрашивает у пользователя ключ (например, 'brand', 'model' и т.д.)

Выводит соответствующее значение из словаря

Если введенного ключа нет в словаре, выводит сообщение "Key not found"

Обрабатывает случай, когда пользователь вводит ключ в неправильном регистре (например, 'Brand' вместо 'brand')

Требования:

Используйте input() для получения ключа от пользователя

Реализуйте проверку наличия ключа в словаре

Приведите ввод пользователя к нижнему регистру для унификации
