Улучшите декоратор logger из предыдущей задачи:

Форматирование даты: Используйте формат YYYY-MM-DD HH:MM:SS (без миллисекунд)

Аргументы и ключевые аргументы: Логируйте и *args, и **kwargs

Разделение логики: Разделите запись в файл на две части:

Запись до вызова функции: дата, имя функции, аргументы

Запись после вызова: результат и время выполнения

Добавьте время выполнения: Измерьте и запишите, сколько секунд выполнялась функция

Ожидаемая запись после вызова multiply(3, 4, verbose=True):
2024-01-15 14:30:25 - multiply - args: (3, 4), kwargs: {'verbose': True} - result: 12 - time: 0.000045s

Требования:

Используйте datetime.now().strftime("%Y-%m-%d %H:%M:%S") для форматирования даты

Измеряйте время выполнения с помощью time.time() (import time)

Результат и время выполнения добавляйте в ту же строку лога
