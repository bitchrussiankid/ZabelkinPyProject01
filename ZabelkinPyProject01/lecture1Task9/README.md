Напишите программу, которая демонстрирует переопределение переменной на основе её собственного значения и сравнения.

- Создайте переменную score и присвойте ей значение 100.

- Выведите на экран сообщение: "Начальный счет: [значение score]".

- Пользователь вводит количество очков, которые нужно добавить. Сохраните это значение в переменную bonus.

- Переопределите переменную score, добавив к ней значение bonus.

- Проверьте, стал ли новый score больше 150.

- Выведите на экран сообщение: "Новый счет: [значение score]".

- Выведите на экран результат проверки (True или False) на вопрос "Счет превысил 150?".
