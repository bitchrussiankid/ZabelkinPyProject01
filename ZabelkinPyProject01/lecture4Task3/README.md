Дан список температур за неделю: temperatures = [25, 30, 18, 22, 28, 35, 20]

Напишите программу, которая выводит True, если выполняются ОБА условия:

В любой из дней недели температура была выше 32 градусов

В любой из дней температура была ниже 20 градусов

В противном случае выведите False.

Требования:

Используйте логические операторы (and, or)

Не используйте циклы for или while (работайте напрямую со списком)

Для проверки условий используйте встроенные функции Python для работы со списками
