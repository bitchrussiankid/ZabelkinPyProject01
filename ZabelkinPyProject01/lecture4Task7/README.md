Дан список чисел: numbers = [15, 7, 22, 9, 36, 11, 28, 42, 5, 19]

Напишите программу, которая:

Создает новый список, содержащий элементы с четными индексами (0, 2, 4...) из первой половины исходного списка

Создает второй список, содержащий элементы с нечетными индексами (1, 3, 5...) из второй половины исходного списка

Объединяет эти два списка

Если сумма элементов полученного списка больше 50, выводит "Large sum", иначе - "Small sum"

Требования:

Используйте срезы и индексацию

Не используйте циклы

Используйте условные операторы

Учтите случай, когда список имеет нечетную длину
