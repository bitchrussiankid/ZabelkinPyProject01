Создайте список books, содержащий не менее 5 названий книг (строки). Напишите программу, которая запрашивает у пользователя номер книги (от 1 до количества книг в списке) и выводит название этой книги.

Требования:

Обязательно обработайте случай, если пользователь введет номер, выходящий за границы списка (меньше 1 или больше длины списка)

В случае ошибки выведите сообщение "Invalid book number"

Учтите, что пользователь вводит номер, начиная с 1, а индексы в списке начинаются с 0
