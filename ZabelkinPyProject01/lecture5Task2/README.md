Дан словарь:

country_capitals = {
    'Germany': 'Berlin',
    'Italy': 'Rome', 
    'France': 'Paris',
    'Spain': 'Madrid'
}

Напишите программу, которая создает три отдельных списка:

Список всех ключей списка

Список всех значений списка

Список всех пар (ключ, значение) в виде кортежей в списке

А потом формирует первые два списка (ключ, значение) в новый словарь new_country_capitals, который должен быть идентичен оригинальному

Требования:

Используйте соответствующие методы словарей Python

Выведите каждый список на экран

Убедитесь, что порядок элементов соответствует исходному словарю
