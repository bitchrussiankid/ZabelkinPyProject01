Напишите функцию find_word_in_file(filename, word), которая:

Принимает имя файла и слово для поиска

Ищет все вхождения слова в файле (без учёта регистра)

Возвращает список номеров строк, где найдено слово

Если слово не найдено, возвращает пустой список

Дополнительно: Функция должна выводить:

Общее количество найденных вхождений

Каждую найденную строку с её номером

Требования:

Используйте контекстный менеджер with

Читайте файл построчно

Приводите и строку файла, и искомое слово к нижнему регистру

Обрабатывайте случай, когда файл не существует

Пример файла text.txt:

Hello world
Python is great
Hello again
World of python
