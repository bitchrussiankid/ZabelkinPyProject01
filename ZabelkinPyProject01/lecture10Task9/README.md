**Задача 4. Многоуровневое наследование (Employee → Developer → SeniorDeveloper)**

Создай **иерархию классов** для сотрудников IT-компании:

1. **Базовый класс** `Employee` (Сотрудник):
   - Атрибуты: `name`, `salary` (зарплата)
   - Метод: `work()` — возвращает `"Выполняю общие задачи"`

2. **Дочерний класс** `Developer` (Разработчик), наследует от `Employee`:
   - Добавляет атрибут: `programming_language` (язык программирования)
   - **Переопределяет** `work()` — возвращает `"Пишу код на [язык]"`

3. **Внучатый класс** `SeniorDeveloper` (Старший разработчик), наследует от `Developer`:
   - Добавляет атрибут: `team_size` (размер команды, которой руководит)
   - **Переопределяет** `work()` — возвращает `"Руковожу командой из [team_size] человек и пишу код"`

**Требования:**
- Каждый конструктор вызывает родительский через `super().__init__()`
- Продемонстрируй все три уровня

**Пример:**
```python
emp = Employee("Иван", 50000)
dev = Developer("Анна", 80000, "Python")
senior = SeniorDeveloper("Пётр", 120000, "Java", 5)

print(emp.work())    # Выполняю общие задачи
print(dev.work())    # Пишу код на Python  
print(senior.work()) # Руковожу командой из 5 человек и пишу код
```
