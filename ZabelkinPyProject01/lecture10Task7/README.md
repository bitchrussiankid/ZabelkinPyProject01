**Задача 2. Переопределение методов (Shape → Circle)**

Создай **родительский класс** `Shape` (Фигура):
- Метод: `area()` — возвращает `0` (площадь по умолчанию)

Создай **дочерний класс** `Circle` (Круг), который наследует от `Shape`:
- Атрибут: `radius` (радиус)
- **Полностью переопределяет** метод `area()` — должен возвращать площадь круга по формуле:  
  `π * radius²` (используй `math.pi` или `3.14`)

**Требования:**
1. Используй `import math` для точного значения π
2. Площадь округли до 2 знаков после запятой

**Пример:**
```python
shape = Shape()
circle = Circle(5)

print(shape.area())  # 0
print(circle.area()) # 78.54 (если π=3.14)
```
