Задача от лида андроид-разработки
Суть задачи:
Необходимо написать алгоритм, который позволит: раздать числа из массива пользователю со следующими условиями:
1. Каждая итерация - одно случайное выданное число
2. "Раздать" - значит числа должны закончиться

Алгоритм при этом должен быть оптимизированным под высокую нагрузку, т.е. не занимать слишком много памяти и времени на выполнение
