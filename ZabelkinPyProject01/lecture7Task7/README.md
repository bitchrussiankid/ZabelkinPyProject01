Создайте файл emails.txt со следующим содержимым:

john@gmail.com
alice@yahoo.com
bob@gmail.com
carol@hotmail.com
dave@gmail.com
eve@yahoo.com

Напишите программу, которая:

Считывает содержимое файла emails.txt

Ищет все email-адреса, которые содержат @gmail.com

Записывает найденные адреса в новый файл gmail_addresses.txt (каждый с новой строки)

Выводит на экран:

Количество найденных gmail-адресов

Сам список найденных адресов

Сообщение "Gmail addresses saved to file"

Требования:

Используйте контекстный менеджер with для обоих файлов

Читайте файл построчно для экономии памяти

Убедитесь, что файл gmail_addresses.txt создается/перезаписывается при каждом запуске
