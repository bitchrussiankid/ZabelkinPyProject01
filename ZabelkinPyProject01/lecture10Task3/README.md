Задача 3. Дополнение методов (расширение через super())

Создай родительский класс Employee (Сотрудник):

Атрибуты: name (имя), employee_id (ID сотрудника)

Метод: get_info() — возвращает "Сотрудник: [имя], ID: [ID]"

Создай дочерний класс Manager (Менеджер), который наследует от Employee:

Добавляет атрибут: department (отдел)

Расширяет метод get_info():

Сначала вызывает родительский метод get_info()

Затем добавляет ", Отдел: [отдел]" в конец строки

Используй super().get_info() для получения базовой информации

Требования:

В методе Manager.get_info() используй super() для вызова родительского метода

Не дублируй код из Employee.get_info()

Добавляй только новую информацию
