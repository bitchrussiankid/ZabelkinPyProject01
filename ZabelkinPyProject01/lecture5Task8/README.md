Создайте два словаря:

dict1 = {'a': 1, 'b': 2, 'e': 5}
dict2 = {'c': 3, 'd': 4, 'e': 6}

Напишите программу, которая:

Объединяет dict2 в dict1 с помощью метода update() (изменяя исходный dict1)

Создает новый словарь dict3, который содержит все элементы из обоих словарей (без изменения исходных)

Очищает словарь dict1

Выводит все три словаря после операций

Требования:

Продемонстрируйте разницу между изменяющим объединением (update) и созданием нового словаря

Покажите работу метода clear()

Убедитесь, что при конфликте ключей приоритет отдается словарю dict2
