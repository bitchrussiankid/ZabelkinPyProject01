Создайте текстовый файл example.txt со следующим содержимым:
Hello, World!
This is a test file.
Python is awesome.
Working with files is fun.

Напишите программу, которая:

Открывает файл example.txt для чтения

Считывает все содержимое файла сразу

Выводит содержимое на экран

Закрывает файл автоматически с помощью контекстного менеджера with

Требования:

Используйте режим чтения 'r'

Примените метод read() для получения всего содержимого

Убедитесь, что файл закрывается автоматически
