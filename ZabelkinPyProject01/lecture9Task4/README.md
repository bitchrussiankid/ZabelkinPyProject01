Создайте класс Student (Студент) и несколько объектов этого класса.

Требования:

Класс должен иметь конструктор с параметрами:

name (имя, строка)

age (возраст, целое число)

course (курс, целое число)

Создайте три разных объекта (студента) с разными данными

Для каждого объекта выведите информацию в формате:

Student 1: Name - [имя], Age - [возраст], Course - [курс]
Student 2: Name - [имя], Age - [возраст], Course - [курс]
Student 3: Name - [имя], Age - [возраст], Course - [курс]
