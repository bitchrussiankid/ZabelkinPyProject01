Создайте файл fruits.txt со следующим содержимым:

apple
banana
orange
grape
melon


Напишите программу, которая:

Открывает файл fruits.txt для чтения

Читает файл построчно с помощью цикла

Для каждой строки выводит: "Fruit X: [название фрукта]"

В конце выводит: "Total fruits: X"

Требования:

Используйте контекстный менеджер with

Читайте файл непосредственно в цикле (не используйте readlines())

Удалите лишние пробелы и переносы строк у каждой строки

Подсчитайте общее количество фруктов
