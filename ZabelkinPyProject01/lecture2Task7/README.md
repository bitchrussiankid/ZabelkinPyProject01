Напишите функцию extract_alternating_chars(s), которая принимает строку s и возвращает новую строку, составленную из символов исходной строки, взятых через один, начиная с первого символа (то есть символы на позициях 0, 2, 4 и т. д.).

Например:

для строки 'Python' результат должен быть 'Pto';

для строки 'Hello, World!' — 'Hlo ol!'.
