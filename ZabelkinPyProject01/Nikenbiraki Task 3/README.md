Напишите функцию rotate_and_mask(s, n), которая принимает строку s и целое число n, и выполняет следующие действия:

Циклически сдвигает строку вправо на n позиций (если n отрицательное — сдвиг влево).
Например, сдвиг 'abcde' на 2 вправо → 'deabc'; на −1 влево → 'bcdea'.

Заменяет в полученной строке:

все гласные (a, e, i, o, u, в любом регистре) на символ *;

все цифры — на символ #;

остальные символы оставляет без изменений.

Возвращает итоговую строку.

Примеры:

rotate_and_mask('Hello123', 1) → '3H*ll###'
(сдвиг на 1 вправо: '3Hello12' → замена: '3H*ll###')

rotate_and_mask('Python', -2) → 'th*nPy'
(сдвиг на 2 влево: 'thonPy' → замена: 'th*nPy')

Учтите:

строка может быть пустой;

n может быть больше длины строки или отрицательным;

регистр гласных важен только для определения замены (заменяется и A, и a).
