Улучшите класс Book из предыдущей задачи, добавив конструктор __init__.

Требования:

Конструктор должен принимать три параметра: self, title, author

При создании объекта значения должны сразу присваиваться атрибутам:

self.title = title

self.author = author

Создайте два объекта, передавая значения прямо при создании:

book1 = Book("1984", "George Orwell")

book2 = Book("To Kill a Mockingbird", "Harper Lee")
