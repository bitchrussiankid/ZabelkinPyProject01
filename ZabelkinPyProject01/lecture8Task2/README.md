Напишите декоратор timer, который:

Измеряет время выполнения любой функции

Выводит время в формате: "Function 'имя_функции' executed in X.XXX seconds"

Работает с функциями, которые принимают любые аргументы

Сохраняет возможность вызова функции с её оригинальными аргументами

Тестовая функция:

def example_function(n):
    total = 0
    for i in range(n):
        total += i
    return total

Требования:

Используйте модуль time (import time)

Декоратор должен возвращать результат оригинальной функции

Выводить время выполнения после завершения функции
