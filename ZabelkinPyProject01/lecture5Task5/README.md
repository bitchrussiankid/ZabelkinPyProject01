Написать программу, которая:
1. Принимает у пользователя логин
2. Смотрит в "БД", присутствует ли такой пользователь в системе (Если нет - выводит "User not found")
3. Принимает у пользователя пароль
4. Смотрит в "БД", правильный ли пароль.
   "БД" хранит захэшированный пароль, поэтому перед тем, как сравнить пароль, который указал пользователь - его надо захэшировать по тем же правилам.
5. Если пароль подходит - смотрит в "БД", какая роль у пользователя и выводит ее в формате "Role: [роль]"
6. Проверяет, есть ли в "БД" хотя бы один пользователь с ролью "Модератор":
- Если есть - выводит "Moderator exists in the system"
- Если нет - выводит "No moderator in the system"
