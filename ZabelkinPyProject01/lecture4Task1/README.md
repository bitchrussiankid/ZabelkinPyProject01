Напишите программу, которая запрашивает у пользователя число. Если число положительное, выведите "Positive", если отрицательное — "Negative", если ноль — "Zero".

Дополнительное условие: Учтите случай, когда пользователь может ввести не число (например, текст). В этом случае выведите сообщение "Error: please enter a numeric value".

Требования:

Используйте try-except для обработки некорректного ввода

Используйте условные операторы if/elif/else для проверки числа

Программа должна обрабатывать как целые, так и дробные числа
