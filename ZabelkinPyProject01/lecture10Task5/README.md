**Задача 5. Множественное наследование**

Создай **два независимых родительских класса**:

1. **Класс `Engine` (Двигатель):**
   - Атрибут: `power` (мощность в л.с.)
   - Метод: `get_power_info()` — возвращает `"Мощность: X л.с."`

2. **Класс `Radio` (Радио):**
   - Атрибут: `has_radio` (есть ли радио, булево значение)
   - Метод: `play_music()` — если `has_radio` True, возвращает `"Играет музыка"`, иначе `"Радио отсутствует"`

Теперь создай **класс `Car`**, который наследует от **обоих родителей**:
- Добавляет атрибуты: `brand`, `model`
- **Переопределяет** метод `get_power_info()` — добавляет к родительской информации `" для [brand] [model]"`

**Требования:**
1. Используй множественное наследование: `class Car(Engine, Radio):`
2. Конструктор `Car` должен инициализировать атрибуты всех родителей
3. Покажи, как работают методы обоих родителей

**Пример:**
```python
car = Car(brand="Toyota", model="Camry", power=150, has_radio=True)

print(car.get_power_info())  # Мощность: 150 л.с. для Toyota Camry
print(car.play_music())      # Играет музыка
```

**Подсказка:** В конструкторе `Car` нужно вызвать конструкторы обоих родителей:
```python
def __init__(self, brand, model, power, has_radio):
    Engine.__init__(self, power)    # или super().__init__(power) 
    Radio.__init__(self, has_radio) # но с множественным наследованием сложнее
    self.brand = brand
    self.model = model
```
