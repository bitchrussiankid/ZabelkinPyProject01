**Задача 3. Расширение через super() (Device → SmartDevice)**

Создай **родительский класс** `Device` (Устройство):
- Атрибуты: `name` (название), `power` (потребляемая мощность в ваттах)
- Метод: `get_info()` — возвращает `"Устройство: [название], Потребление: [power] Вт"`

Создай **дочерний класс** `SmartDevice` (Умное устройство), который наследует от `Device`:
- Добавляет атрибут: `is_connected` (подключено ли к интернету, булево значение)
- **Расширяет** метод `get_info()`:
  - Сначала получает информацию от родителя через `super()`
  - Затем добавляет `", Подключено: Да/Нет"` в зависимости от `is_connected`

**Требования:**
1. Используй `super().get_info()` для получения базовой информации
2. Не дублируй код из `Device.get_info()`

**Пример:**
```python
device = Device("Лампа", 60)
smart = SmartDevice("Умная лампа", 10, True)

print(device.get_info())  # Устройство: Лампа, Потребление: 60 Вт
print(smart.get_info())   # Устройство: Умная лампа, Потребление: 10 Вт, Подключено: Да
```
