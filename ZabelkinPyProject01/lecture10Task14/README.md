### **Задача №2: Композиция и специальные методы**

**Темы:** Классы, объекты, методы, `__init__`, `__str__`, композиция

Создай класс `Task`:
1. Атрибуты: `title` (строка), `description` (строка), `completed` (булево, по умолчанию `False`).
2. Метод `toggle_completed` — меняет состояние `completed` на противоположное.
3. Метод `__str__` — возвращает строку вида: `"[✓] Title: Description"` если задача выполнена, или `"[ ] Title: Description"` если не выполнена.

Создай класс `TaskList`:
1. Атрибут: `tasks` (список объектов `Task`).
2. Метод `add_task` — добавляет задачу в список.
3. Метод `complete_task_by_title` — принимает строку (название задачи), находит первую задачу с таким названием и помечает её как выполненную.
4. Метод `__str__` — возвращает строку со всеми задачами (каждая с новой строки), и в конце строку `"Total: X/Y completed"`, где X — количество выполненных задач, Y — общее количество задач.

**Дополнительное условие (декоратор):**
Создай декоратор `log_activity`, который принимает строку-действие (например, `"add"` или `"complete"`) и выводит сообщение в формате: `"[LOG] Action: <действие>, Task: <название задачи>"`. Примени этот декоратор к методам `add_task` и `complete_task_by_title`.

**Задание:**
1. Реализуй классы и декоратор.
2. Продемонстрируй работу:
   - Создай список задач
   - Добавь несколько задач (сработает лог)
   - Отметь одну задачу выполненной (сработает лог)
   - Выведи весь список через `print(task_list)`
