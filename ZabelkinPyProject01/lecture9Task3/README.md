Добавьте в класс Book метод display_info().

Требования:

Метод должен принимать только self в качестве параметра

Метод должен возвращать строку в формате: "Название by Автор"

Метод не должен сам выводить текст, только возвращать строку

В основной программе вызовите этот метод для каждого объекта и выведите результат
