Создайте класс Book (Книга) с двумя атрибутами:

title (название, строка)

author (автор, строка)

Требования:

Класс должен быть пустым (используйте ключевое слово pass)

Создайте два экземпляра (объекта) этого класса:

book1 с названием "1984" и автором "George Orwell"

book2 с названием "To Kill a Mockingbird" и автором "Harper Lee"

Выведите атрибуты каждого объекта

Пример вывода:

Book 1: 1984 by George Orwell
Book 2: To Kill a Mockingbird by Harper Lee

Подсказка: После создания объектов, присвойте значения атрибутам напрямую (пока без конструктора).
