Создайте пустой список shopping_list. Затем выполните следующие действия:

Запросите у пользователя ввести три товара по очереди и добавьте их в конец списка

Замените второй элемент в списке на строку "milk"

Выведите итоговый список на экран

Требования:

Используйте input() для получения данных от пользователя

Используйте метод append() для добавления элементов

Используйте индекс для замены элемента

Учтите, что нумерация элементов начинается с 0
