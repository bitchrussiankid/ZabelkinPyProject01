Цель: Понять, что self хранит состояние конкретного объекта.

Класс Counter:

Атрибут: value (начальное значение 0)

Методы:

increment() — увеличивает value на 1

decrement() — уменьшает value на 1

get_value() — возвращает текущее значение

Требования:

Создайте два разных счётчика (counter1, counter2)

Покажите, что они независимы:

Увеличьте первый счётчик 3 раза

Уменьшите второй счётчик 2 раза

Выведите значения обоих счётчиков


Помним, что self - обозначает удержание значения конкретного объекта класса!
