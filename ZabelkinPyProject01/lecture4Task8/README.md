Дан список оценок студента: grades = [85, 92, 78, 90, 65, 88, 76]

Напишите программу, которая определяет статус студента по следующим правилам:

"Excellent" - если ВСЕ оценки выше 80 И средний балл выше 85

"Good" - если все оценки выше 70 И хотя бы одна оценка выше 90

"Average" - если нет оценок ниже 60 И количество оценок выше 80 больше половины

"Need improvement" - если есть хотя бы одна оценка ниже 70

"Fail" - если есть хотя бы одна оценка ниже 60

Приоритет правил: Если студент подходит под несколько категорий, выбрать самую высокую (Excellent > Good > Average > Need improvement > Fail)

Требования:

Используйте логические операторы и условные операторы

Не используйте циклы (работайте со встроенными функциями для списков)

Учтите приоритет условий
