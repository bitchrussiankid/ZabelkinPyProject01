Дан словарь с оценками студентов:

grades = {
    'Alice': [85, 92, 78],
    'Bob': [65, 70, 80],
    'Charlie': [90, 85, 95],
    'Diana': [75, 80, 70],
    'Eve': [88, 92, 96]
}

Напишите программу, которая:

Для каждого студента вычисляет средний балл

Сохраняет средние баллы в новый словарь average_grades

Используя цикл, находит и выводит:

Имена студентов с средним баллом выше 85

Общее количество таких студентов

Для студентов с средним баллом ниже 75 выводит предупреждение: "Внимание: [имя] имеет низкий средний балл: [балл]"

Требования:

Используйте цикл for для перебора словаря

Применяйте методы словаря для доступа к значениям

Используйте условные операторы внутри цикла
