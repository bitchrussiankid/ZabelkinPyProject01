Напишите декоратор logger, который:

Записывает в файл function_log.txt информацию о каждом вызове функции

Логируемые данные:

Дата и время вызова (в формате YYYY-MM-DD HH:MM:SS)

Имя вызванной функции

Переданные аргументы (в виде строки)

Возвращаемое значение

Каждая запись в файле — с новой строки

Декоратор должен добавлять записи в файл, а не перезаписывать его

Ожидаемая запись в файле после вызова multiply(3, 4):
2024-01-15 14:30:25 - multiply - args: (3, 4) - result: 12

Требования:

Используйте модуль datetime для получения текущего времени

Открывайте файл в режиме добавления ('a')

Сохраняйте возможность вызова функции с её оригинальными аргументами
