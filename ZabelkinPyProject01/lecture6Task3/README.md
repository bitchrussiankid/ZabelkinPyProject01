Дан словарь с продажами менеджеров за неделю:

sales = {
    'Ivanov': {'mon': 120, 'tue': 135, 'wed': 140, 'thu': 130, 'fri': 125},
    'Petrov': {'mon': 90, 'tue': 110, 'wed': 95, 'thu': 105, 'fri': 115},
    'Sidorov': {'mon': 150, 'tue': 140, 'wed': 160, 'thu': 145, 'fri': 155}
}

Напишите программу, которая:

Для каждого менеджера вычисляет общую сумму продаж за неделю

Находит средние продажи по каждому дню недели (среднее между всеми менеджерами)

Выводит:

Имя менеджера с наибольшими общими продажами

День недели с наибольшими средними продажами

Общий отчет в читаемом формате

Требования:

Используйте вложенные циклы для работы со словарем словарей

Применяйте методы словарей для доступа к данным

Сохраняйте промежуточные результаты в новые словари
