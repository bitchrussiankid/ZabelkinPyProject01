### **Задача №1: Наследование и декораторы**

**Темы:** Наследование (`super()`), методы класса, декораторы с аргументами

Создай базовый класс `SmartDevice`:
1.  В `__init__` должны быть атрибуты `device_id` (строка) и `status` (по умолчанию `"off"`).
2.  Метод `toggle_status`, который меняет `status` на противоположный (`"on"` ↔ `"off"`) и возвращает новое состояние.
3.  Метод `get_info`, который возвращает строку: `"Device [device_id] is [status]"`.

Создай класс-наследник `SmartLight`:
1.  В `__init__` также принимает параметр `brightness` (целое число, по умолчанию 50). Используй `super()` для вызова конструктора родителя.
2.  Добавь атрибут `max_brightness = 100`.
3.  Переопредели метод `get_info`, чтобы он возвращал строку вида: `"Light [device_id] is [status] with brightness [brightness]/[max_brightness]"`.

**Декоратор:**
Создай декоратор с **аргументом** `min_brightness`, который проверяет, что устанавливаемая яркость у экземпляра `SmartLight` не меньше этого порога. Если яркость ниже, декоратор должен выводить предупреждение: `"Warning! Brightness below minimum allowed ([min_brightness]). Set to [min_brightness]."` и устанавливать яркость равной `min_brightness`. Примени этот декоратор к методу, который устанавливает яркость (тебе нужно будет придумать и добавить такой метод в класс `SmartLight`).

**Задание:**
1.  Реализуй классы и декоратор.
2.  Продемонстрируй работу:
    *   Создай экземпляр `SmartLight`.
    *   Вызови `get_info`.
    *   Включи/выключи устройство (используя унаследованный метод).
    *   Продемонстрируй работу декоратора, попытавшись установить яркость меньше минимального порога.
